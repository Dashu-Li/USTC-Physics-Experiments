\begin{abstract}
本实验以核磁共振(NMR)原理为基础，通过在恒定磁场中施加射频脉冲，观测样品的自由感应衰减(FID)与自旋回波信号，测定核磁共振频率及弛豫时间 $T_1$ 与 $T_2$。本报告依据实验手册（核磁共振.pdf）整理实验原理与方法，数据部分留空，待实验测量后填入并完成数据处理与讨论。
\end{abstract}

\section{实验题目}
实验3-9：核磁共振（NMR）实验

\section{实验目的}
\begin{enumerate}
    \item 理解核磁共振的基本物理机制与共振条件。
    \item 掌握使用脉冲射频激发与检测自由感应信号（FID）和回波信号的方法。
    \item 测定样品的共振频率及弛豫时间 $T_1$、$T_2$，并进行误差分析。
\end{enumerate}

\section{实验原理}
本节依据实验手册对核磁共振的理论要点做简要归纳，便于后续数据处理与讨论。

\subsection{核自旋与磁矩}
原子核具有自旋角动量 $\mathbf{I}$，相应的磁矩为 $\boldsymbol{\mu}=\gamma\mathbf{I}$，其中 $\gamma$ 为回旋比。置于静磁场 $B_0$ 中，能级发生塞曼分裂，系统在平衡时的纵向磁化强度为 $M_0$。

\subsection{拉莫尔进动与共振条件}
磁矩在 $B_0$ 下以拉莫尔角频率进动：
$$\omega_0=\gamma B_0,\quad f_0=\omega_0/2\pi.$$ 
当施加的横向射频场频率与 $f_0$ 共振时，可激发横向磁化并在接收线圈中产生可测电压信号（FID 或回波）。

\subsection{弛豫过程与测量方法}
横向弛豫时间 $T_2$ 描述相干分量的衰减，纵向弛豫时间 $T_1$ 描述磁化恢复。常用测量方法包括：
\begin{itemize}
    \item 自由感应衰减（FID）直接记录激发后信号随时间的衰减；
    \item 反转恢复（inversion-recovery）序列用于测量 $T_1$；
    \item 自旋回波（spin-echo）或 CPMG 序列用于测量 $T_2$，回波幅度随延时 $\tau$ 指数衰减 $A(\tau)=A_0e^{-2\tau/T_2}$。
\end{itemize}

在数据处理时常采用指数拟合、傅里叶变换（频谱分析）与谱线宽度互证等方法。

\section{实验步骤（概括）}
\begin{enumerate}
    \item 将样品安装到射频线圈中心，固定位置。
    \item 打开磁体、调节匀场（必要时做shimming），记录主磁场 $B_0$。
    \item 设置射频频率在预计共振附近，扫描频率以确定共振峰 $f_0$。
    \item 确定 90° 与 180° 脉冲宽度（通过脉冲归一化或方位调谐）。
    \item 采用 FID、反转恢复与自旋回波序列分别测量共振信号与弛豫过程，记录原始波形数据。
    \item 将测得的时域信号进行指数拟合或逆变换以求取 $T_1$, $T_2$ 及谱线宽度等参数。
\end{enumerate}

\section{数据处理与计算（方法与公式）}
\subsection{频谱与回旋比}
对频率扫描数据取频谱峰位 $f_0$，并记录对应的 $B_0$ 值，可计算回旋比：
$$\gamma=\frac{2\pi f_0}{B_0}.$$ 
若样品为已知核种（如质子），该关系可用于 $B_0$ 的互证或标定。

\subsection{弛豫时间拟合方法}
\begin{itemize}
    \item $T_2$：采用自旋回波序列，回波幅度随延时 $\tau$ 指数衰减，拟合模型为 $A(\tau)=A_0e^{-2\tau/T_2}$；拟合得到 $\hat{T}_2$ 及标准误差 $u(T_2)$。
    \item $T_1$：采用反转恢复序列，纵向磁化恢复遵循 $M_z(t)=M_0(1-2e^{-t/T_1})$，拟合得到 $\hat{T}_1$ 与不确定度 $u(T_1)$。
\end{itemize}

\subsection{谱线宽度互证}
谱线半高宽 $\Delta f$ 与横向弛豫时间之间近似关系为
$$T_2\approx\frac{1}{\pi\Delta f},$$
可作为 $T_2$ 的互证方法。

\subsection{不确定度评估}
拟合参数的不确定度可由最小二乘拟合的协方差矩阵给出；必要时对残差进行自助法(bootstrapping)估计，最终结果按置信区间给出（例如 $k=2$ 时近似 95\% 置信）。
